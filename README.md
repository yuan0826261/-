# 四.小專題題目:21點
## 遊戲介紹:使手中牌的點數之和等於或接近 21 點，且不能超過 21 點，同時要比莊家的點數大。
###　1.從2到10按照面值計算(2就是2點，5就是5點)，J、Q、K一律記為10點，而A是最靈活的牌，可記為1或11點。 
###　2.玩家在發牌前決定籌碼下注。
3.玩家獲得兩張明牌(正面朝上)，而莊家得到一張明牌一張暗牌(一張正面朝上，另一張正面朝下)。
4.玩家可以選擇要牌或停牌，要牌可以再拿一張牌，試圖接近21點，若超過21點則視為爆牌，立即判輸。停牌則不再拿牌，保持當下點數。
